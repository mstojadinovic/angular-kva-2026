O Projektu

Projekat je izrađen kao deo kursa "Klijentske Veb Aplikacije" na Univerzitetu Singidunum. Cilj projekta je bio da se kreira web aplikacija koja omogućava korisnicima da pronađu i rezervišu igračke.

Korišćene tehnologije:

  • Angular 21
  • Angular Material
  • Sweet Alerts 2
  • Axios HTTP Client
  • Toys API
